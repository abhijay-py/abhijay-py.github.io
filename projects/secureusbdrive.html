    Secure USB Drive

    A multi-bank USB drive secured by fingerprint and password.

    This project is a device that makes it possible for four seperate users to utilize the same drive to access seperate data.

    This project was created as a part of the ECE 47700 senior design course at Purdue University alongside Brandon Liu, Stanley So, and Joshua Wai.

    The team created a device that takes in input of a fingerprint and password to allow users access to their data on a USB drive.

    The product was created from scratch, with the individual components being selected and prototyped, the circuitry being modeled and PCB designed in KiCAD, the components soldered onto the board, the microcontroller programmed with the firmware for the product, and the casing being modeled for the device.

    The drive is controlled by a STM32H7 microcontroller, utilizing a LCD screen for HUD, a keypad to recieve password input, a fingerprint sensor to recieve fingerprint data, and a SPI NAND flash to store data.

    Some errors were made during production of this project due to the short one semester timeline that was given to the team. Some notes were taken for a potential V2 to be manufactured in the future, such as minimizing the footprint of some of the user facing parts to shrink the size of the device.

    The source code, CAD models of the casing, the schematics and PCB design, and a user manual can be found here here.

    Datasheets, journals, and reports and more information not located on the Github directly can be found on the website stored in the website folder of the github repository.
