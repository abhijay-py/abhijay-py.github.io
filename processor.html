    RV321 Processor

    A Model of a RV321 Processor

    This project is still under development.

    This project is an implementation of a RISC-V processor that handles most RV321 Base Integer Instructions using SystemVerilog.

    All RV321 Base Integer Instructions listed here were implemented apart from the unsigned operations and the enviornment instructions. The RTL Diagrams for each part were drawn up before the system was implemented.

    This processor operates by manually entering the machine code of RISC-V assembly instructions manually into the instruction memory in the SystemVerilog files and then allowing the processor to run through the instructions.

    Structures such as the instruction memory and data were simplified down to store 256 Bytes each instead of the roughly 4GB they can normally store to simplify coding and processing. Any attempt to access memory outside these bounds will result in a zero being returned. Also, overflow errors and other errors will not be handled (it assume all instructions will run perfectly).
